# Transfer Funds from Sellers

After refunding a customer, you may need to retrieve funds from a seller back into your account using a refund transfer. Each refund may have exactly **one** refund transfer, whose total amount cannot exceed the refund amount.

## Getting Started

<GettingStartedSection />

### Get your Secret Key

Next you will need your `Secret Key`. [Go to your Developers section](https://portal.publicsquare.com/developers/api-keys) and click `Reveal` for your `Secret Key` and copy the value.

<Alert>Save the Secret Key as it will be used in the next steps of this guide.</Alert>

## Prerequisites

This guide assumes you have already completed:

- [Refund Payments](/guides/refunds/refund-payments) — you will need a refund ID.
- [Transfer Funds to Sellers](/guides/marketplaces/transfer-funds-to-sellers) — you will need the payment transfer ID and item IDs from the original transfer.

Grab the refund `id`:

```json showLineNumbers title="Refund"
{
  // highlight-next-line
  "id": "rfd_2YKewBonG4tgk12MheY3PiHDy",
  "account_id": "acc_B518niGwGYKzig6vtrRVZGGGV",
  "environment": "test",
  ...
}
```

And the payment transfer `id` and each item `id` from the original transfer:

```json showLineNumbers title="Payment Transfer"
{
  // highlight-next-line
  "id": "trnf_7yFLQWACr3DYDSz1xpoEAVfdq",
  "account_id": "acc_B518niGwGYKzig6vtrRVZGGGV",
  "environment": "test",
  ...
  "items": [{
    // highlight-next-line
    "id": "trfi_7yFLQWACr3DYDSz1xpoEAVfdq",
    "seller_account_id": "acc_w6dogDaHuU6h1N5e5vfXLUYf",
    "amount": 1000,
    "transfer_fee": 100
  }]
}
```

## Create a Refund Transfer

Call the [Create Refund Transfer](/api/financial/create-refund-transfer) endpoint:

```shell showLineNumbers title="Create a Refund Transfer"
curl 'https://api.publicsquare.com/refund-transfers' \
-X 'POST' \
-H 'X-API-KEY: <SECRET_API_KEY>' \
-H 'Content-Type: application/json' \
-d '{
  // highlight-next-line
  "refund_id": "rfd_2YKewBonG4tgk12MheY3PiHDy",
  // highlight-next-line
  "payment_transfer_id": "trnf_7yFLQWACr3DYDSz1xpoEAVfdq",
  "items": [{
    // highlight-next-line
    "payment_transfer_item_id": "trfi_7yFLQWACr3DYDSz1xpoEAVfdq",
    "amount": 1000,
    "transfer_fee": 100
  }]
}'
```

<Alert>The amount is provided in cents. 1000 is the equivalent of $10.00</Alert>

You should see a refund transfer result similar to:

